# Customers Churn Analysis

## Project Overview
This project focuses on analyzing customer churn to identify key factors that influence customer retention and churn behavior.

## Objective
To understand churn patterns based on contract types, payment methods, tenure, and customer demographics, and to provide actionable business recommendations.

## Dataset
- Customer Churn.csv

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Repository Files
- `Customer Churn.csv` – Dataset used for analysis
- `TCA.ipynb` – Jupyter Notebook with analysis and visualizations
- `Summary and Recommendations.pdf` – Key findings and business insights

## Key Insights
- Month-to-month contract customers have the highest churn rate
- Electronic check users churn more compared to other payment methods
- Customers with less than one year tenure are more likely to churn
- Long-term contracts significantly improve retention

## Conclusion
The analysis highlights the importance of long-term contracts, early customer engagement, and optimized payment methods to reduce churn.
